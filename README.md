# times-project

## Intro

- New York Times Open API 를 활용해서 뉴스 검색 서비스 제작하였습니다.

## Stack and Tools

- React.js
- React icons
- emotion
- Redux
- Redux Toolkit

## Design

- Adobe XD

## Description

### 뉴스 검색 기능

- 단어를 검색하면 해당 단어를 포함하는 뉴스를 보여준다.

### 뉴스 즐겨찾기 기능

- 뉴스를 즐겨찾기 하면 즐겨찾기 한 뉴스들만 모아서 볼 수 있다.
- 검색어 태그 필터 기능 (검색어 태그를 클릭하면 해당 검색어 기사들을 필터링해 보여준다)
