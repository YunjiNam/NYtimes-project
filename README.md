# times-project

## Intro

- New York Times Open API 를 활용해서 뉴스 검색 서비스 제작하였습니다.
- 반응형 웹

## Stack and Tools

- React.js
- React icons
- classNames
- Scss
- Redux
- Redux Saga

## Design

- Adobe XD

## Description

### 뉴스 검색 기능

- 단어를 검색하면 해당 단어를 포함하는 뉴스를 보여준다.

### 뉴스 즐겨찾기 기능

- 뉴스를 즐겨찾기 하면 즐겨찾기 한 뉴스들만 모아서 볼 수 있다.
